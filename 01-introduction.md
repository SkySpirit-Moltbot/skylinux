# Lesson 1: Introduction à Linux

## Qu'est-ce que Linux ?

Linux est un **système d'exploitation** gratuit et open-source. Il est utilisé partout :
- Sur les serveurs web (97% des serveurs)
- Sur les smartphones Android
- Sur les supercalculateurs
- Sur ton Raspberry Pi !

## Pourquoi apprendre Linux ?

| Avantage | Description |
|----------|-------------|
| Gratuit | Pas de licence à payer |
| Libre | Tu peux voir et modifier le code |
| Sécurité | Moins de virus que Windows |
| Demandé | Très recherché sur le marché du travail |
| Communautaire | Une grande communauté pour t'aider |

## Les principales distributions

Une **distribution** (ou "distro") c'est une version de Linux. Les plus populaires :

| Distribution | Pour qui ? |
|--------------|------------|
| **Ubuntu** | Débutants, desktop |
| **Debian** | Serveurs, utilisateurs avancés |
| **Fedora** | Développement,桌面 |
| **Arch Linux** | Pour les experts |
| **Raspberry Pi OS** | Raspberry Pi |

## Les bases de la console

Ouvre un terminal. Voici quelques commandes essentielles :

```bash
# Naviguer dans les dossiers
ls              # Lister les fichiers
cd dossier      # Entrer dans un dossier
cd ..           # Revenir en arrière
pwd             # Voir où tu es

# Créer et supprimer
mkdir nom       # Créer un dossier
rm nom          # Supprimer un fichier
rm -r nom       # Supprimer un dossier

# Manipuler les fichiers
cp source dest  # Copier
mv source dest  # Déplacer / Renommer
cat fichier     # Lire un fichier
```
